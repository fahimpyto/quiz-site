তোমার error log পরিষ্কারভাবে দেখাচ্ছে আসল সমস্যা কী।
স্ক্রিনশটের popup (`Failed to submit quiz`) আসছে কারণ backend এ **500 error** হচ্ছে।

মূল error 👇

```
django.db.utils.OperationalError: no such table: quiz_quizattempt
```

এর মানে:

👉 Django database এ **QuizAttempt table তৈরি হয়নি**
কিন্তু তোমার view এ এটা ব্যবহার করা হচ্ছে:

```python
QuizAttempt.objects.filter(...)
```

তাই server crash করছে।

---

# ✅ Fix (২টা command)

Terminal এ এই দুইটা command চালাও:

```bash
python manage.py makemigrations
python manage.py migrate
```

---

### কেন লাগছে?

তুমি নতুন model add করেছ:

```python
class QuizAttempt(models.Model):
```

কিন্তু database এ তার table এখনো create হয়নি।

`makemigrations` → migration file বানাবে
`migrate` → database table create করবে
